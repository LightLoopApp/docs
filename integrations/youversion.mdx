LightLoop pulls scripture from **two sources**, both in the same translation picker — used in the Media Library **Add Bible** flow and the [Sermon Builder](/web-app/sermons):

- **YouVersion** — popular licensed English translations, listed at the top of the picker.
- **Free Use Bible** — a free, open library of **1,200+ translations across 1,000+ languages**, available below YouVersion as a searchable option.

Search any passage, choose a translation, and customize the slide.

## Free Use Bible (all languages)

Below the YouVersion presets, the translation picker includes **Free Use Bible (all languages) — search…**, powered by the [Free Use Bible API](https://bible.helloao.org). It offers **1,200+ public-domain and openly-licensed translations across 1,000+ languages** — many you won't find anywhere else.

To use it:

1. Open the translation picker and choose **Free Use Bible (all languages) — search…**
2. Type a translation **name, abbreviation, or language** (e.g. `Berean`, `Spanish`, `Swahili`)
3. Pick a result — your choice is remembered for next time
4. Enter your passage reference as usual and search

<Tip>
  If a YouVersion translation throws an error (some are intermittently unavailable), the Free Use Bible source is a reliable fallback — and the only way to display many non-English languages.
</Tip>

## Adding a Bible slide

1. In the **Media Library**, click **Add Bible**
2. Type a passage reference — e.g. `John 3:16`, `Psalm 23`, or `Romans 8:28-30`
3. Select a **translation** — a YouVersion preset, or **Free Use Bible (all languages) — search…** for any other language/translation
4. The verse text appears in the preview — review it before saving
5. Customize the slide style (see below)
6. Click **Save** — the slide is added to your library as a Bible item

### Multi-verse passages

If you enter a range (e.g. `Psalm 23:1-6`), LightLoop fetches all verses and combines them into a single slide. Long passages are automatically split across multiple slides — each slide shows a natural break in the text.

## Style options

Bible slides share the same style engine as Songs:

| Option | Description |
| --- | --- |
| **Font** | Family, size, weight, italic, all-caps |
| **Color** | Text color |
| **Alignment** | Horizontal (left / center / right) and vertical (top / middle / bottom) |
| **Drop shadow** | Offset, blur, color |
| **Text stroke** | Width and color |
| **Text background** | Color + opacity panel behind the text for contrast over busy backgrounds |
| **Background** | Image or solid color behind the verse text |

The reference and translation abbreviation are displayed automatically in a smaller badge below the verse text.

## Using Bible slides in playlists

Bible items behave like any other media item — add them to a playlist, reorder with drag-and-drop, and set how long each slide stays on screen before auto-advancing.

In the **iOS Remote app**, Bible items in Control Mode expand verse by verse so the operator can advance through each slide manually during a sermon.

## Caching behavior

Bible slides are rendered server-side the first time they appear on an Apple TV. The rendered image is then cached locally so the slide plays even without an internet connection. If the slide style or passage is edited, the cache is refreshed automatically.
